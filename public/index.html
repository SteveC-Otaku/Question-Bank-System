        Question Management Help
        Overview

        This page allows you to manage your question bank. You can add, edit, view, and delete questions, as well as add detailed answers and explanations.

        Filtering & Search
        • Search: Type keywords to search through question titles and content
        • Subject Filter: Filter questions by subject area
        • Type Filter: Filter by question type (Multiple Choice, True/False, etc.)
        • Difficulty Filter: Filter by difficulty level (Easy, Medium, Hard)
        • Clear Filters: Reset all filters to show all questions
        Actions Column

        The Actions column contains buttons for each question. Here's what each button does:

        View

        View complete question details including content, options, and answers

        Edit

        Modify question content, type, difficulty, or other properties

        Add/Edit Answer

        Add detailed explanations, solutions, key points, and references

        Delete

        Permanently remove the question from your database

        Adding New Questions
        1. Click the "Add Question" button in the top-right corner
        2. Fill in the question details:
          • Title: A brief title for the question
          • Subject: The subject area (e.g., Mathematics, Science)
          • Type: Choose from Multiple Choice, True/False, Short Answer, Programming, or Essay
          • Difficulty: Set the difficulty level
          • Content: The actual question text
        3. For Multiple Choice questions, add options and mark the correct one
        4. For True/False questions, select the correct answer
        5. For Short Answer questions, provide the correct answer text
        6. Add relevant tags to help with organization
        7. Click "Save Question" to add it to your database
        Adding Answers

        After creating a question, you can add detailed answers and explanations:

        • Explanation: A brief explanation of why the answer is correct
        • Detailed Solution: Step-by-step solution process
        • Key Points: Important concepts or formulas to remember
        • References: Sources, textbooks, or materials for further study
        • Answer Difficulty: How difficult the answer explanation is
        Navigation

        Use the pagination controls at the bottom of the table to navigate through multiple pages of questions. The system shows 10 questions per page by default.

        User Account Management (Admin Only)

        Administrators have access to comprehensive user management features:

        • View All Users: See a complete list of all registered users
        • Create New Accounts: Add new users with specific roles
        • Edit User Information: Update user details, roles, and permissions
        • Activate/Deactivate Accounts: Enable or disable user access
        • Reset Passwords: Generate new passwords for users
        • Role Management: Assign and change user roles (Student, Teacher, Admin)
        • Account Deletion: Remove user accounts when necessary
        Tip: You can use the search and filter functions together to quickly find specific questions. For example, search for "algebra" and filter by "Mathematics" to find all algebra-related math questions.
